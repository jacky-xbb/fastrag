# 示例查询

不知道问啥时，从这里挑。每条都对应库里某篇具体标准，能真打中检索。
问法贴近「标准号 + 表名/指标名 + 页码」，命中越准。
也可以**不带标准号、用产品名或大白话问**（块锚点已含产品名、正文按条款切），见第六节。

## 一、查某标准某表的技术指标（最常用）

- GBT 18242-2025 弹性体塑性体改性沥青防水卷材，I 型卷材的可溶物含量要求是多少？
- GBT 23457-2017 预铺防水卷材的拉力和钉杆撕裂强度要求是多少？
- GBT 35467-2017 湿铺防水卷材的不透水性怎么规定的？
- JCT 974-2005 道桥用改性沥青防水卷材，低温柔度要求是多少度？
- JT/T 536-2018 路桥用塑性体改性沥青防水卷材的耐热性指标？

## 二、试验方法 / 检验规则 / 在第几页

- 沥青防水卷材的撕裂性能（钉杆法）怎么测？在哪个标准哪一页？（→ GBT 328.18-2007）
- GB/T 328.18 钉杆法撕裂试验用多大直径的尖钉？拉伸速度多少？（→ GBT 328.18-2007）
- 高分子防水卷材接缝剥离性能的试验步骤？（→ GBT 328.21-2007）
- GB/T 328.21 接缝剥离性能试验的原理是什么？（→ GBT 328.21-2007）
- 防水卷材吸水性怎么测？取样和判定规则在第几页？（→ GBT 328.27-2007）
- GB/T 328.27 吸水率怎么计算？取几块试件的平均值？（→ GBT 328.27-2007）
- 沥青和高分子防水卷材抗冲击性能的试验方法？（→ GBT 328.24-2007）
- 高分子防水卷材撕裂性能（梯形/割口法）的试验原理？（→ GBT 328.19-2007）
- 沥青防水卷材接缝剪切性能试验，夹具间距是多少？（→ GBT 328.22-2007）

## 三、历史/旧版标准（当普通文档检索，不再特殊标注，见 ADR-0005）

- 氯化聚乙烯-橡胶共混防水卷材标准的指标？（→ jc 684-1997）
- jc 684-1997 的拉伸强度要求是多少？

## 四、行业专用（铁路 / 水运 / 公路）

- 铁路桥梁混凝土桥面防水层有哪些性能要求？（→ TB/T 2965-2018）
- 铁路隧道防水板的厚度和拉伸性能要求？（→ TB/T 3360.1-2023）
- 水运工程防水卷材试验规程里，低温柔性怎么测？（→ JTS/T 232-2019）
- JTS/T 232 钢筋弯曲试验的步骤？试验后看什么、记录什么？（→ JTS/T 232-2019）
- 沥青加铺层抗裂贴的剥离强度要求？（→ JT/T 971-2015）

## 五、产品范围 / 分类 / 术语 / 前言（非指标表，问"是什么/分几类/代替谁"）

- GBT 18242 的防水卷材按胎基分为哪两类？（→ 聚酯毡 PY / 玻纤毡 G）
- GBT 23457 预铺防水卷材适用于与什么粘结？（→ 自粘胶与后浇混凝土）
- GBT 23457-2017 预铺防水卷材代替了哪个旧标准？（前言，→ GB/T 23457—2009）
- JT/T 536 路桥用塑性体改性沥青防水卷材的胎基和改性剂是什么？（术语定义）
- 沥青加铺层抗裂贴按施工方式分成哪几类？（→ JT/T 971，热粘型 / 自粘型）
- 湿铺防水卷材的型式检验在哪些情况下要做？（→ GBT 35467）

## 六、不带标准号也能问（用产品名 / 大白话，靠块锚点召回）

- 高分子防水卷材接缝剥离性能的试验步骤？（→ GBT 328.21-2007）
- 防水卷材吸水性试验怎么做？要刷掉什么、用什么吸干？（→ GBT 328.27-2007）
- 湿铺防水卷材用在什么工程？靠什么和基层粘结？（→ GBT 35467-2017）
- 道桥用改性沥青防水卷材适用在什么地方？（→ JC/T 974-2005）
- 铁路桥梁混凝土桥面防水层适用于什么桥面？（→ TB/T 2965-2018）
- 防水卷材抗冲击穿刺试验的原理是什么？用什么做支撑？（→ GBT 328.24-2007）
- 氯化聚乙烯橡胶共混防水卷材适用于什么卷材？（→ jc 684-1997）

## 七、库内没有 → 触发联网兜底（验证 webSearchTool）

下面这些是**防水卷材域内、但当前库里没入**的标准（库内 18 篇见上）。属于域内问题，
agent 库内检索不到 → 触发 `webSearchTool` 联网搜索，结果应标注「来源：联网」。
若日后把它们 ingest 进库，则改由库内回答（联网仅作兜底）。

- GB/T 12952-2011 聚氯乙烯（PVC）防水卷材的拉伸强度要求是多少？（库内无 → 联网）
- GB/T 12953-2003 氯化聚乙烯防水卷材的低温柔性要求？（库内无；注意与库内 JC 684-1997「共混」卷材区分）
- GB 23441-2009 自粘聚合物改性沥青防水卷材的可溶物含量要求？（库内无；与库内 GB/T 18242 弹性体/塑性体不同）

反例（**离域**，应礼貌拒答、**不**联网）：南京天气、今天新闻——本助手只答防水卷材国标/行标。
